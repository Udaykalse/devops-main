# 🚀 DevOps Learning Journey (Hands-on POCs)

This repository documents my **daily hands-on DevOps learning journey** through practical Proof of Concepts (POCs).

The goal of this repo is to learn DevOps **by building and experimenting**, not just theory.

Each folder represents a daily learning module covering real-world DevOps fundamentals including Linux, Git, CI/CD, Docker, and Cloud concepts.

---

## 📌 Objectives

* Understand DevOps fundamentals step by step
* Practice real industry workflows
* Build an automation mindset
* Learn deployment & troubleshooting
* Maintain structured daily progress

---
